Actualizar

Permite actualizar las puntuaciones de las canciones y de las listas inmediatamente, sin tener que esperar a que la base de datos lo haga. Este proceso se lleva a cabo automaticamente todos los dias.
Este proceso calcula la puntuacion Semanal de las listas y de las canciones,tomando los datos obtenidos por las votaciones de los usuarios en la ultima semana.
Nota: Este proceso puede durar mucho tiempo. No se debe usar en horas con gran afluencia de usuarios.
